How to import a vendor-specific Radius dictionary into Cisco ISE.

Problem
-------

Import a Radius dictionary.

Solution
--------

1. Download the radius dictionary.
2. Import the dictionary file.
3. Configure profiles with the dictionary information.

Implementation
--------------

Download the radius dictionary from either the vendor website or third-party sources. Cisco community offers a wide range of dictionaries for downloading.

Import the file downloaded into Cisco ISE:

1. From the management GUI, go to `Policy > Policy Elements > Dictionaries`
2. From the Dictionaries panel, expand `System > Radius` and select `Radius Vendors`
3. Click `Import`
4. Select the file path and click `Import`

The dictionary information is now available:

1. Go to `Policy > Policy Elements > Results`
2. Select `Authorization > Authorization Profiles`
3. Click `Add`
4. Under `Advanced Attributes Settings`, select the new dictionary to view all its attributes
